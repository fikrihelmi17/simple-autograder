# Minimal Requirement

**Python: 3.10.12**  
**Package Manager: pip**  
**Virtual Environment: venv**  
**Node.js: 12.22.9**
**Linux**

===============================

# Set Up the Project
`python3 -m venv .venv`  
`source .venv/bin/activate`   
`pip install -r requirement.txt`   
`chmod +x main.py`   

# Run The Program
`./main.py`
